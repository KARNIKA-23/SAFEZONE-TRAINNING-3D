
# SAFEZONE TRAINNING 3D

## Overview

SAFEZONE TRAINNING 3D is a 3D training simulation project designed to demonstrate safe-zone awareness and training scenarios in an interactive environment. The project focuses on visual learning through a simulated 3D space rather than theoretical explanations.

This repository is intended for **learning, experimentation, and academic demonstration purposes**.
